ssh -i ~/.ssh/k8s_cluster k8sadmin@192.168.11.203


CONFIGURE SSH FOR EASY ACCESS:
-------------------------------
Add to ~/.ssh/config:

Host k8s-control
    HostName 192.168.11.201
    User k8sadmin
    IdentityFile ~/.ssh/k8s_cluster

Host k8s-worker-01
    HostName 192.168.11.202
    User k8sadmin
    IdentityFile ~/.ssh/k8s_cluster

Host k8s-worker-02
    HostName 192.168.11.203
    User k8sadmin
    IdentityFile ~/.ssh/k8s_cluster

Then connect simply with:
    ssh k8s-control
    ssh k8s-worker-01
    ssh k8s-worker-02
